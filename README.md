# datathon-project
데이터 분석 부트캠프 내에서 진행된 데이터톤 프로젝트로, 외식업 데이터를 활용한 비즈니스 컨설팅 분석을 수행하여 대상 수상

📊 데이터톤 프로젝트 – 외식업 비즈니스 컨설팅 데이터 분석 (대상 수상)
프로젝트 개요

본 프로젝트는 가상의 외식업체 전문 비즈니스 컨설팅 회사를 가정하고,
외식업 데이터를 기반으로 의사결정에 활용 가능한 지표와 인사이트를 도출하는 것을 목표로 진행한 데이터톤 프로젝트입니다.

진행 형태: 팀 프로젝트 (데이터톤)

기간: 2025.10.20 – 2025.11.18

팀원 수: 4명

성과: 데이터톤 대상 수상

프로젝트 주제

가상의 외식업체 전문 비즈니스 컨설팅 회사를 가정한 데이터 기반 컨설팅 분석

데이터 출처

Yelp Open Dataset

담당 역할 및 기여

팀 내에서 각자 지표를 정의하고 이를 통합해 최종 컨설팅용 API 설계를 목표로 프로젝트를 진행했습니다.
저는 다음 역할을 수행했습니다.

데이터 수집 및 전처리

EDA 설계 및 분석 방향 설정

핵심 지표 정의 및 가설 설정

가설 검증을 위한 시각화 제작

인사이트 정리 및 발표 자료(PPT) 구성

개인 기여 비율: 약 20%

팀 프로젝트 특성상 전체 내용을 단독으로 수행하지는 않았으나,
본인이 담당한 지표 영역에서는 분석 설계부터 인사이트 도출까지 전 과정을 주도적으로 수행했습니다.

문제 정의

외식업 분석에서 단순 매출 지표를 넘어,
컨설팅 관점에서 실제 의사결정에 활용 가능한 복합 지표를 정의하고
이를 통해 외식업체 운영 및 전략 수립에 도움이 되는 인사이트를 도출하고자 했습니다.

분석 접근 방식

Yelp Open Dataset 구조 및 데이터 품질 파악

주제에 맞는 핵심 지표 정의

지표 기반 가설 설정

시각화를 통한 가설 검증

컨설팅 관점의 인사이트 도출

지표를 통합한 최종 API 설계 방향 제시주요 인사이트

주제에 부합하는 지표를 중심으로 가설을 설정하여
단순 현황 분석이 아닌 원인 중심의 분석이 가능했음

지표 간 관계를 종합적으로 해석하여
외식업 컨설팅에 바로 활용 가능한 시사점 도출

분석 결과를 API 형태로 확장 가능한 구조로 설계하여
실무 적용 가능성 측면에서 높은 평가를 받음

대상 수상 배경 (개인 관점)

문제 정의 → 지표 설계 → 가설 → 인사이트로 이어지는
분석 흐름이 처음부터 끝까지 일관되게 구성됨

주제에 적합한 가설 설정과 명확한 인사이트 도출

분석 결과를 비즈니스 컨설팅 관점에서 해석했다는 점에서
높은 평가를 받았다고 판단합니다.

한계 및 개선 방향

데이터톤 특성상 제한된 시간과 데이터 환경에서 분석 수행

실제 서비스 데이터 및 실시간 데이터 연동 시
지표 및 분석 고도화 가능

사용 기술

Python (Pandas)

데이터 전처리 및 EDA

가설 기반 데이터 분석

데이터 시각화
